## D. Ciclo de Conversão de Capital (OBRIGATÓRIO)

Dados do último ITR: estoques, contas a receber (CR), contas a pagar (CP), receita anualizada.

```
PMR = (CR / Receita) × 365
PME = (Estoque / CMV) × 365
PMP = (CP / CMV) × 365
Ciclo Operacional = PMR + PME
Ciclo de Caixa = CO − PMP
```
